Drill Reimbursement

PA DOH EMSO, using funds from a HRSA grant, is providing limited reimbursements for EMS agencies who participate in certain tabletop or field exercises.

A full description of this program and required forms, are available here:

  • Drill Reimbursement Program description (note deadlines)
  • Drill Reimbursement Program invoice form*
  • IRS Form W-9 (PDF) Request for Taxpayer Identification Number and Certification (with instructions)*

*Complete these two forms only, and submit them to EMSI.
