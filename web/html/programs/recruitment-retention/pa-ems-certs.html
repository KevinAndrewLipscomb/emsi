Finding courses

You can view a current list of certification courses using the EMSAMS system.  Here's how:

  • Anyone, including any member of the general public, is welcome to register for an EMSAMS account.  To find out how, read our EMSAMS page.

  • Once you log into EMSAMS, do the following.  (Consider printing these instructions first.)

    1. Click the "Continuing Education/Certification Course Applications" link.

    2. On the left, in the "Search" section, click "Search for classes being held".

    3. On the Class Search page, scroll to the bottom and press the "More Search Criteria" button.

    4. Set the fields as follows:

      Classes starting between:
      County where the course is being held:
      Region where the course is being held:
      Course code:       		Set as desired
      All other fields Leave blank
       

    5. Scroll back up to the top of the page and click the "Search Now" button -- then wait.

Registering for courses

To register for any of these courses, call or write to the person listed in EMSAMS as the course Contact.  EMSI does not handle registration for these courses.
