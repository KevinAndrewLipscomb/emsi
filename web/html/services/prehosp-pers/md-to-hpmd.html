MD to HP Physician process

» We process applications from physicians wishing to attain the DOH Health Profession Physician certification, which allows the physician to function as a member an ambulance crew.

To begin the process, contact Nancy Theilacker (right).

NOTE:  Parts of this process cannot be conducted online.  For instance, the Student Application form must be completed with a Number 2 pencil, and must be fed manually into an OpScan™ machine.  Where appropriate, we have provided other parts online so that you can start completing them now.

The definitive way to get this process rolling is to contact Nancy Theilacker.
