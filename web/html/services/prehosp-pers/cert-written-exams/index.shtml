Certification Written Examinations

EMSI conducts written exams for the following state certifications at EMSI headquarters, as described in this section:
  • First Responder
  • EMT
  • Basic Vehicle Rescue
  • Basic Rescue Practices
  • Special Vehicle Rescue

Preparatory Instructions:
  1. Register ahead of time by calling EMSI (see phone numbers at the top of this page).  Have the following information ready when you call:
    • Your full name
    • Which session you want to attend
    • Which certification you wish to be tested for
    • Your birthdate
    • Your Social Security Number
    • Your daytime phone number
    • Your state EMS certification number (if applicable)
    • Your state EMS certification expiration date
    • Reason you want to take the exam (challenge, reciprocity, first time taking, retest)
    • Where you took the class (If challenging, where you took the practical)
    • Month and year the class ended
    • Class number (if known)
    • FR and EMT students:  Do you need to retest any practical stations?
  2. Bring photo ID.
  3. Arrive 15 minutes prior to start time.
    No one will be admitted after the start of the exam.
