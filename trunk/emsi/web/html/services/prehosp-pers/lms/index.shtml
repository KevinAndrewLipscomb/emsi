You must fax the completion certificate to your regional EMS council to get credit for taking the course.

 

Here's some terminology you should know, especially if you call for help:

LMS
Learning Management System.  A generic name for any self-paced instructional web site.
 
PaHealthInfo.net
The web address of the LMS provided by the Pennsylvania Department of Health.
 
EMSEd
The company that the PA DOH has hired to run PaHealthInfo.net.
 
CentreLearn
The technology that EMSed has chosen to implement PaHealthInfo.net.
