FRED

Establishment guidelines

The following guidelines are for receiving hospitals, PSAPs, EMAs, and other organizations wishing to establish a Continuous FRED Monitoring Site at their facility.

Administrative/operational considerations

  1. Specific employee roles must be designated, empowered, informed, and available to react to alerts by entering the appropriate resource availability information into FRED.

  2. There is no limit to the number of computers per facility on which FRED may run.

  3. The FRED monitoring computer may be used for any number of other applications.

  4. Suitable locations for Continuous FRED Monitoring Sites include:

    • Emergency Department nurse's station
    • Emergency Department ambulatory triage room
    • Emergency Department registration desk
    • Security monitoring station
    • Switchboard operator's station
    • Pharmacy
    • Nurse administrator's office
    5.  

  5. Regardless of location, you should establish a FRED SOP that designates someone in charge of submitting resource data to FRED in case of an alert.  This person may be:

    • the Charge Nurse
    • the on-call Nurse Administrator
    • the Pit Boss
    • etc.
    6.  

  6. FRED can now issue alerts directly to alphanumeric pagers and text-enabled cellular telephones.

Technical considerations

  1. Identify a computer system administrator to help implement this section.

  2. Identify a computer that:

    • Runs Internet Explorer;
    • Is always on;
    • Has an always-on connection to the Internet;
    • Is located within earshot of an always-staffed area;
    • Has a reasonably loud internal or external speaker set.

    The computer may be used for any number of other applications beside FRED.

  3. There is no limit to the number of computers per facility on which FRED may run.

  4. Assure that the computer, its network connection, and its browser are configured to allow the following:

    • Unrestricted access to, and JavaScript execution on, <http://*.miemss.org:80/*>;
    • Continuously inactive logins (although screensaver locking is acceptable);
    • META REFRESH
    5.  

  5. Put a shortcut to FRED in the following places for All Users:

    • Startup folder
    • Desktop
    • Favorites folder
