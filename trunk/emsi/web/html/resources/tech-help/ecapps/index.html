Background

On EMSAMS, you can check your Con Ed status and much more!

Also, tasks previously accomplished using the Ambulance Service License Application form, Class Registration Application form or the New Course Approval Application form must now be accomplished via EMSAMS.

Accessing the new site

EMSI recommends you access EMSAMS via the page you are reading now.  On this page, we will provide you with hints and tools for using EMSAMS.

Visiting the site the first time

When you first visit EMSAMS, you must use the "Register" button to set up your own login name and password.

"I've registered.  What next?"

"I'm having trouble!"

  • Con Ed Sponsors and Training Institutes should see the User Guide.

  • Instructions for all other tasks are being developed.

  • Did you click the "Register" button and set up your own username and password?

  • If EMSAMS says you need 128-bit encryption, either use a newer computer or download and install the High Encryption Pack from Microsoft.

  • When registering a new class, only enter numbers in the Student Cost field.  EMSAMS may crash if you enter any other characters.

  • When registering a new class, make the Subject Of Class brief.  EMSAMS may crash on long entries.
